xpbcopy and xpbpaste web clipboard

1. Make sure you have curl or wget

2. Add xpbcopy and xpbpaste bash functions to your ~/.bash_profile:

curl {{url}}/bash >> ~/.bash_profile

3. Copy (example):

cat /etc/hosts | xpbcopy

4. Paste:

xpbpaste <pbid>
